2025年11月某个深夜突然意识到自己还没有坐过头等舱。自己没有的东西总是充满着吸引力，只是我甚至连积分票都不清楚应该怎么换。因为被美国航司们折磨了几年，所以对全日空的服务印象特别深刻，目标就定在 JFK - HND 这条航线上的 NH 159/109。

11月8号(T-30)在系统里候补了 12月8号 NH 159 的特典头等舱。然后意识到我需要一个保底机票以防出不了特典头等舱。我第一个想法是定一个现金票，选择了 AA 167 (JFK - HND) 价格 589 USD 的经济舱，但是退款只能退 AA 的 travel credit，很不爽。但后来反应过来我其实根本不需要现金票，马上在24小时内把票退了，在 ANA 特典官网换了一张同个日期 NH 159 经济舱的确认票作为保底，这样日本就肯定能去了。

接着开始排大队，把 NH 159 先后 3 个航班的所有商务舱和头等舱都加到了候补队列，打算实在不行就在纽约机场睡一天，我最会睡机场了。聊天机器人查到 12月8号 NH 159 的特典头等舱候补队列只有 1 个人（算上我 2 个），我据此以为转正概率挺大的，后来行程结束后继续多方调查才意识到我太天真了，虽然还是给我坐到了，乐。

{% asset_img waitlist_clear.png This is an image %}

在11月17号(T-21)候补转正了时间最合适的 NH 159。可以坐上头等舱了！网上别人说 ANA 的候补就是拿来伤人心的，事后看来名不虚传，除了这个以外剩下的全都没排到。难道真的是运气好？于是我把这个航班的特典经济舱退了，里程-3000.

不过实在担心前序航班的时间出差错，所以还是定了当天下一班（12个小时后）出发的 NH 109 的特典超经作为应急，到时赶不上就在机场退了 NH 159 特典头等舱坐 NH 109 超经。这样搭配后应该是一定是能去到日本的，因为到纽约的航班可太多了，而且大家都清楚12月的纽约绝没有到会因为气象条件停飞的程度。到纽约无非是成本和时间，只是我是来占便宜的所以确实最好是两个都不花。

本来以为赶上了就可以把 NH 109 特典超经改签到远期，不用扣退票费，事实证明我还是太乐观了。 


## 行程

周末我在 Atlanta, GA 开会。周日(T-1)中午刚散会，我就到 ATL 搭乘到 JFK 的通勤航班 DL 2466 （计划 17:50 - 20:22）了。我还是非常担心美国国内线的延误让我错过凌晨的 NH 159，虽然我给自己安排了 4h 的 layover，但我很不相信美国人的时刻，所以原本打算早点到机场然后用 Delta 的 same-day stand-by 尽可能改签到更早航班，但是折腾半天实在没搞懂怎么操作，似乎要加一笔巨大的票差才能 stand-by，于是放弃。而且我的机票是会议主办方替我订的，我能做的改动十分受限。这也算是礼宾待遇了，但我的卡还怎么玩呢！

实际上后来我发现我目前所有航班至多只延误 2h13m，所以似乎有些紧张过头了。不安真的是天生的毛病。

结果证明我还是对美国航空业有很根本的理解的——就是延误了。这次我的票总算不是基础经济舱，于是总算获得了用 AmEx Platinum 进入休息室的权利。但是当我看到前序航班已经延误了 5 个小时我就觉得麻烦大了，果然预期起飞时间在 monitor 开始左右横跳，登机口开始左右横跳，我的心率也开始左右横跳。好在原定到起飞时间时飞机已经来到附近了。最后在 20:57 到达 JFK。

{% asset_img DL2466.png This is an image %}

这下完全不急了。慢慢下飞机然后坐 AirTrain 前往 T7。全日空在 JFK 接待我的地勤是位三哥，但是一直全程一直敬礼和敬语着实让我受到了不小冲击。一个是我没指望在美国有这种服务，另一个是用英语说日式敬语（我忘了具体说了什么了）真的非常别扭！查 eVisa 时看到旁边一家人掏出了人数两倍的蓝绿相间的护照，嗯。

{% asset_img bordingpass.png This is an image %}

大概是因为没什么人的缘故，T7 安检没有优先通道，但还是丝滑进场。JFK T7 里的不是 ANA 自营的休息室，很难对美国人抱什么期待。第一次进到了信用卡和 PP 卡不能进的地方。

{% asset_img doorway.jpg This is an image %}

吃的虽然比美国其他航司好了，但还是乏善可陈。感觉这些菜品说是印度航空休息室都能信。唯一的特点大概就是人是真的少，三四个区域只有四个人在用。

{% asset_img food.png This is an image %}

去隔壁热闹的商务舱顺了点这边没有的吃的，被头等舱休息室这边的工作人员问明明有吃的怎么还过去（笑）。接着再去隔壁洗了个澡（头等舱休息室没有洗澡差评），回来时新到的前台员工还想拦住我不给我进。之后歇了一会，思考改签 NH 109 的那张超经的票，但是五月和更加往后都没有票，我也不想折腾滚雪球，感觉扣 3000 里程取消算了（疼！）。就差不多上机了。因为动作收拾东西慢了，去到时已经开始商务舱登机了，没体验到最优先登机，有一点遗憾。

其实我还想吃素咖喱和海鲜盖饭的，但真的吃不下了，哎！

在飞机上这段时间我本来是要在学校监考的，但是直接溜走去享受人生了嘿嘿。考试前学生们肯定不放过我，虽然在享受人生时真的不想理学生但我也尽可能回邮件了。一想到接下来明明在度假还要抽一天在酒店改胡言乱语满纸荒唐言的卷我就想一拳打爆地球。

到了 HND T2。下飞机前问空乘沐浴间是在入国审查前还是后，然后空乘「え——」地想了一会然后跟我说不知道。这个「え——」实在是 responsive 得有点不习惯，虽然知道在日本 being responsive 是礼貌，心里总是在吐槽明明不是非出声不可的。第二个下机但是第一个人走转机去了，所以我是第一个过入国审查的。出税关后找了一会沐浴间，前台接待可能起床气还没消，跟空乘们相比态度冷淡得怀疑是不待见外国人。洗了个澡后，在四楼的 Bic Camera 拿了预约的 sim 卡，就去上野跟朋友会合了。

说起来全日空执行这条航线的是 B777[（座位图点这里)](https://www.ana.co.jp/www2/travel-information/seat-map/b-777-300ER-n212map.jpg)，可以看到有一半的空间是商务舱，挺壮观的……



其实无论是吃的还是服务很多我都没太明白。一些小红书的意见说，头等舱比商务舱好在洗手间完全不用排队，确实这是我印象最深能切实感受到的点。因为并没有明白，所以短期内我应该也不会再花这么多积分坐头等舱了，毕竟只是想吃的话下飞机我能吃光东京。

总共花费的成本是 85k+3k+3k=91k ANA miles 和税费 27160 JPY。市场价按 10k USD 计的话，有 11 cents/mile，算是本人玩卡史上一个突出成就了。
